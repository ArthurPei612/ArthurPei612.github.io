2D Vector Simulation Engine

This project provides a Python-based simulation engine to calculate and visualize the dot product and cross product (z-component) of two-dimensional vectors.

Features

  • Calculates the dot product (X · Y).
  • Calculates the 2D cross product (X × Y = x1*y2 - x2*y1).
  • Visualizes the input vectors X and Y originating from the origin.
  • Displays the parallelogram formed by X and Y.
  • Provides geometric interpretations of the dot product (angle, projection) and cross product (signed area, orientation).

How to Run

To run the simulation and see example visualizations:

  1. Ensure you have Python installed on your system.
  2. You will also need the matplotlib and numpy libraries. If you don't have them, install them using pip:
    pip install matplotlib numpy
  3. Navigate to the root directory of this project in your terminal.
  4. Run the visualizer script:
    python visualizer.py

The script will output information to the console and display a series of plots, each illustrating a different test case with two vectors, their products, and the geometric interpretations.

The core vector calculations are in vector_engine.py, and the visualization logic is in visualizer.py. You can modify the example vectors in visualizer.py to test with your own inputs.
